# Actividad JavaScript - DOM y Math

## Descripción
Se desarrollan ejercicios de manipulación del DOM, uso de querySelector, eventos y funciones del objeto Math.

## Archivos
- index.html → DOM básico
- app.js → lógica + Math
- query.html → ejercicios avanzados
- app2.js → interacción con botones
- app.css → estilos

## Ejecución
1. Abrir index.html
2. Presionar F12 → consola
3. Para ejercicios avanzados, entrar a query.html

## Temas
- DOM (id, clase, etiqueta)
- querySelector / querySelectorAll
- Eventos (addEventListener)
- Math (random, pow, sqrt)
