# Actividad JavaScript - Fundamentos

## 📌 Descripción
Este proyecto contiene una serie de ejercicios básicos en JavaScript enfocados en:

- Tipos de datos primitivos
- Operaciones matemáticas
- Manipulación de strings
- Uso de template literals
- Ejecución de código en la consola del navegador

---

## 📁 Estructura del proyecto

actividad-js/
│
├── index.html
└── app.js

---

## ⚙️ Ejecución

1. Abrir el archivo `index.html` en el navegador
2. Presionar `F12`
3. Ir a la pestaña **Consola**
4. Ver los resultados del programa

---

## 🧠 Contenido desarrollado

### 1. Tipos primitivos
Se declararon variables de los siguientes tipos:
- Number
- String
- Boolean
- Undefined
- Null
- Symbol
- BigInt

Todos los valores se muestran en consola.

---
