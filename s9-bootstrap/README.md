Las Custom Properties (o variables CSS) en Bootstrap, introducidas a partir de Bootstrap 5, son una funcionalidad clave que permite personalizar el framework en tiempo real. Estas variables incluyen colores del tema,breakpoints, fuentes y espaciados. 

¿Qué son? Son variables CSS nativas accesibles en todo el documento y desde JavaScript. Bootstrap las utiliza para facilitar la modificación de estilos y componentes, permitiendo un diseño responsivo.
Prefijo --bs-: Todas las propiedades personalizadas de Bootstrap incluyen el prefijo --bs- para evitar conflictos con CSS de terceros.
