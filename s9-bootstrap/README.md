# Tarea:

1. Investigar acerca de las [Custom Properties](https://getbootstrap.com/docs/5.3/customize/css-variables/) de Bootstrap
Las propiedades personalizadas (o variables CSS) de Bootstrap son variables que residen en el navegador y permiten modificar el diseño sin necesidad de volver a compilar Sass. Se identifican siempre por el prefijo --bs-.Aquí tienes el desglose de cuáles son y cómo se organizan:
1. Variables Globales (:root): Estas definen los estilos base de toda la aplicación. Se aplican al elemento <html> y afectan a todo el documento.Colores de Paleta: --bs-blue, --bs-red, --bs-primary, --bs-secondary, etc.Fuentes: --bs-body-font-family, --bs-body-font-size, --bs-body-line-height.Colores de Fondo y Texto: --bs-body-color y --bs-body-bg.Bordes: --bs-border-width, --bs-border-style, --bs-border-color, --bs-border-radius.Breakpoints: Variables que indican los puntos de corte (sm, md, lg, xl, xxl).
2. Variables de Componentes: Cada componente de Bootstrap (como botones, alertas o tarjetas) tiene su propio juego de variables. Esto te permite cambiar el estilo de un solo elemento sin afectar a los demás.ComponenteEjemplos de Propiedades PersonalizadasButtons--bs-btn-bg, --bs-btn-border-color, --bs-btn-hover-bgNavbar--bs-navbar-color, --bs-navbar-hover-color, --bs-navbar-brand-padding-yCards--bs-card-spacer-y, --bs-card-bg, --bs-card-border-radiusModals--bs-modal-width, --bs-modal-bg, --bs-modal-margin
3. Variables de Utilidades (Utilities): Bootstrap 5 introdujo variables que alimentan a las clases de utilidad. Por ejemplo, la utilidad de opacidad o de color utiliza variables internas para funcionar:Opacidad: --bs-bg-opacity, --bs-text-opacity.Bordes: --bs-border-opacity.Esto permite hacer cosas como:HTML<div class="p-3 mb-2 bg-primary" style="--bs-bg-opacity: .5;">
  Fondo primario con 50% de opacidad
</div>
4. Variables de Temas (Modo Oscuro): Bootstrap utiliza un conjunto de variables que cambian según el atributo data-bs-theme="dark". Al activar el modo oscuro, las siguientes variables suelen mutar:--bs-body-color-rgb: Cambia a un tono claro.--bs-body-bg-rgb: Cambia a un tono oscuro.--bs-tertiary-bg: Ajusta los colores de fondo de elementos secundarios.
5. Resumen de Ventajas TécnicasManipulación con JS: Puedes cambiar el color de un botón dinámicamente usando element.style.setProperty('--bs-btn-bg', 'purple').Sin Especificidad: No necesitas usar !important para sobrescribir estilos; basta con declarar la variable en el estilo en línea o en una clase hija.Eficiencia: El navegador renderiza los cambios instantáneamente sin procesos de compilación adicionales.
