 2. Variables Globales de Raíz (:root)
Estas afectan a toda la página, incluyendo tu "Hola mundo!!" y la sección de "Huskie":

Colores de Marca:

--bs-primary: El color principal (azul por defecto).

--bs-success: El color para mensajes positivos (verde).

--bs-danger: El color para errores o alertas (rojo).

Cuerpo y Fondo:

--bs-body-bg: El color de fondo de toda la página.

--bs-body-color: El color por defecto de las letras.

Tipografía:

--bs-body-font-family: La fuente que se usa en todo el sitio.

--bs-body-font-size: El tamaño de letra base.
