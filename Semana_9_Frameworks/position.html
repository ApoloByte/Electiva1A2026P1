Posicionamiento CSS

La propiedad position especifica cómo un elemento es posicionado en el documento. Las propiedades top, right, bottom y left determinan la ubicación final de los elementos posicionados.

static relative absolute fixed sticky

Complementadas con top, bottom, left, right y z-index.

Comparación rápida

Valor Flujo normal Referencia de posición Comportamiento con scroll Uso típico
static Sí No aplica Se mueve con el contenido Elementos normales de página
relative Sí (ocupa espacio original) Su posición original Se mueve con el contenido Ajustes, contenedor de absolutos
absolute No Ancestro posicionado más cercano Relativo a su contenedor Badges, tooltips, overlays
fixed No Viewport (ventana del navegador) No se mueve nunca Navbar fija, botón flotante
sticky Sí (hasta el umbral) Ancestro scroll más cercano Se pega al cruzar el umbral Headers, sidebars, tablas

01 — Static

position: static

Valor por defecto de todos los elementos HTML. El elemento sigue el flujo normal del documento sin desplazarse. Las propiedades top, bottom, left, right y z-index no tienen ningún efecto sobre él.

  • Párrafos, encabezados y divs de contenido normal.
  • Elementos que no requieren posicionamiento especial.
  • Para anular un position aplicado por CSS externo.
.elemento {
  position: static; /* valor por defecto */
  /* top, left, right, bottom no tienen efecto */
}

/* Clase equivalente en Bootstrap */
.position-static

Los tres elementos siguen el flujo normal, uno debajo del otro.

02 — Relative

position: relative

El elemento se posiciona según el flujo normal del documento, pero luego se desplaza con relación a sí mismo usando top, left, etc. El espacio original que ocupaba se conserva en el documento. Crea un contexto de posicionamiento para sus hijos absolutos.

  • Ajustar ligeramente la posición visual sin romper el layout.
  • Crear un contenedor de referencia para hijos con absolute.
  • Controlar el z-index de un elemento.
.elemento {
  position: relative;
  top: 20px;   /* baja 20px desde su posición */
  left: 30px;  /* se mueve 30px a la derecha */
}

.position-relative /* Bootstrap */

El elemento azul se desplazó 16px abajo y 24px a la derecha, pero el espacio punteado (posición original) se conserva en el documento.

03 — Absolute

position: absolute

El elemento es removido del flujo normal del documento, sin crearse espacio para él. Se posiciona relativo al ancestro posicionado más cercano (cualquier elemento con position distinto de static). Si no existe ninguno, se ubica relativo al <html>.

  • Badges sobre imágenes o iconos.
  • Tooltips y popovers.
  • Overlays y banners sobre contenido.
  • Elementos decorativos fuera del flujo.
.contenedor {
  position: relative; /* establece el contexto */
}

.hijo {
  position: absolute;
  top: 0;
  right: 0; /* esquina superior derecha */
}

.position-absolute .top-0 .end-0 /* Bootstrap */

04 — Fixed

position: fixed

El elemento es removido del flujo normal. Se posiciona relativo al viewport (la ventana del navegador), y permanece en su lugar aunque el usuario haga scroll. Siempre crea un nuevo contexto de apilamiento. En documentos impresos, el elemento se coloca en la misma posición en cada página.

  • Navbars que permanecen visibles al scrollear.
  • Botones flotantes de acción (FAB).
  • Banners de cookies o notificaciones persistentes.
  • Sidebars fijas en dashboards.
.boton-flotante {
  position: fixed;
  bottom: 2rem;
  right: 2rem;
  z-index: 1000;
}

.position-fixed .bottom-0 .end-0 /* Bootstrap */

05 — Sticky

position: sticky

Híbrido entre relative y fixed. El elemento se trata como relative hasta que cruza un umbral definido con top, momento en el que se fija hasta que alcanza el límite de su contenedor padre. Requiere definir al menos un valor de top, bottom, left o right.

  • Headers de secciones que se pegan al hacer scroll.
  • Navbars que solo se fijan al llegar al tope.
  • Columnas laterales en layouts de dos columnas.
  • Encabezados de letras en listas alfabéticas.
.cabecera {
  position: sticky;
  top: 0; /* umbral: se pega al llegar aquí */
  z-index: 100;
}

.position-sticky .top-0 /* Bootstrap */

Scroll dentro del contenedor para ver cómo cada encabezado se pega al tope mientras su sección está en vista.
